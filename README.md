# TradeTech

TradeTech is designed to solve trading challenges. It allows businesses to manage inventory, monitor pricing, track sales, and automatically calculate profit and loss in real time. It also supports multiple user roles (such as cashier, manager, and director), ensuring controlled access and accountability within the business.
